Рекомендации по оценке: Reference Architecture Fit-Gap Analysis
This estimating guidance relates to performing a reference architecture fit-gap analysis. The estimate depends on the quality of the source reference architecture material and the extent to which the fit-gap needs to be specified.

Основное описание

Performing a reference architecture fit-gap analysis can take between 1 day and 2 weeks.  Factors that influence the effort required include:

  • Whether the reference architecture is selected from a single architectural asset base or involves searching multiple locations.
  • How well documented the reference architecture is and, if two or more are being compared, how consistently they are documented.  In particular how easy is it to identify subject areas covered by the reference architecture?
  • The quality and completeness of the system requirements.
  • The extent to which the statement-of-fit must include the specification of deltas from the asset.  (Typically, requirements for deltas are identified at this point, but not specified in detail until later.)

If the architectural asset being considered is not a reference architecture but is the actual architecture of another similar project, this is likely to increase the effort as it will not be documented for reuse.
