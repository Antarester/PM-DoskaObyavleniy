Рекомендации по оценке: Infrastructure Gap Analysis
This estimating guidance relates to performing an infrastructure gap analysis. The estimate depends on the extent of the existing infrastructure and the number of resources available to perform the analysis.

Основное описание

High Estimate (multi-site, global planning engagement): 1 - 1.5 Weeks with a team of three to four IT Infrastructure architects and client representatives as applicable.

Low Estimate: 1 week with two IT Infrastructure architects and client representatives as applicable.
