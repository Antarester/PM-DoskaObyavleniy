6. Statement of Fit

Provide a summary of the extent of "fit" between the Reference Architecture and customer requirements. For each area where the Reference Architecture does not meet requirements, describe briefly:

  • The unmet requirement.
  • The type and extent of delta that would be requisite for the Reference Architecture to meet the requirement. Should it be extended, modified, or receive a new subject area?
  • An estimate of the cost and risk of the delta.
