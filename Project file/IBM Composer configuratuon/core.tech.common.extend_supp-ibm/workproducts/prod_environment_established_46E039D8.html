Исход: Production Environment Established
This is the required hardware infrastructure, software infrastructure and and integrated subsystem components that are needed to operate the application solution.
Домены: Deployment
Назначение
To provide the infrastructure on which the application will run.
Основное описание
The environment may contain some or all of the following:
  • Network hardware, both WAN and LAN
  • LAN Operating software
  • Network management tools
  • Server hardware
  • Server Operating software
  • System Management Software and monitoring workstations
  • Middleware software
  • Database software
  • Personnel
  • Application solution
Доводка
Последствия отсутствияThe solution cannot be implemented without the Production Environment.
Причины для отказа

This is not needed if the solution does not include an application.
