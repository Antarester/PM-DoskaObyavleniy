Основное описание

Overview

Before creating an iteration record, identify the iteration milestones that have a testing impact. You must create an iteration record for each milestone. Before creating a iteration record, you must create an associated asset registry record.
Note: A configured test case or test suite record cannot be executed unless it is associated with an iteration record

Tool Steps

To create an iteration record:
  1. Log in to a schema repository configured for IBM® Rational® ClearQuest® Test Manager.
  2. Using a supported Rational ClearQuest Client:
    1. From a Rational ClearQuest for Windows client, click Actions > New and from the Choose Record Type window, click TMIteration and then click OK.
    2. From a Rational ClearQuest Web client, click New, select TMIteration, and click Go.
  3. Complete the iteration form:
    1. Type a name for the iteration in the Name field.
    2. Select a Start and End date.
    3. Select an associated asset registry record.
  4. In the Rational ClearQuest for Windows client, click OK. In the Rational ClearQuest Web client, click Save.
After completing the iteration record, you can associate a test plan, test case, configured test case, or test suite record with it.

For more information, see the CQTM online Help topic Creating an iteration record, Using an iteration record, and Modifying an iteration record.
