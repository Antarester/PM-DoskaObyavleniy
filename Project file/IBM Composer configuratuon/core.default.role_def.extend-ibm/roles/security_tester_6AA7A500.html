Роль: Security Tester
This role is responsible for the main security testing activities.
Наборы ролей: Testing
Взаимосвязи
Основное описание

The main focus of this role is testing the Web applications for key security vulnerabilities, verifying the findings, and communicating them to the interested stakeholders. In many cases, the security testing is performed by using an automated testing tool.

Персонал
Квалификация

The person in this role should have these skills:

  • Good understanding of security testing approaches and techniques
  • Good understanding of Web application security 
  • Experience in debugging and diagnostic skills in the security domain
  • Knowledge of the types of systems and applications under test
  • Training in the use of security test automated tools
  • Experience using the security test automated tools
