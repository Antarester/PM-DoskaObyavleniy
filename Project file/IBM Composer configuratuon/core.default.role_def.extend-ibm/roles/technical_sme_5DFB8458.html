Роль: Technical SME
This role is responsible for all the details surrounding the implementation of various aspects of the plan for realizing a solution.
Наборы ролей: Software Capability Improvement
Взаимосвязи
Основное описание
Responsible for all the details surrounding the implementation of various aspects of the plan for realizing a solution.
Персонал
Квалификация

This role must combine these skills:

  • Thorough understanding of the software development domain with an emphasis on the process aspects and metrics
  • Excellent knowledge of the tools that will be part of the solution 
  • Good grasp of the constraints imposed by the environment: technical, business, legal, standards, and so forth 
Подходы к назначениюWithin a project, more people could be assigned to this role, each with just a subset of the technical skills required to implement the solution.
