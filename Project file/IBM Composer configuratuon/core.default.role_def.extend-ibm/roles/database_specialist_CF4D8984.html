Роль: Database Specialist
This role develops and maintains database systems used for storing, analyzing, and retrieving data.
Наборы ролей: Development
Взаимосвязи
Основное описание
This role is responsible for outlining and detailing the logical data model, as well as developing the physical database design. This role also describes how to migrate legacy data source to a target database.
Персонал
Квалификация

A person performing this role should have the following skills:

  • Have logical and analytical skills to solve problems
  • Have interpersonal abilities and communication skills to assess team and organizational needs
  • Know a variety of database systems, in order to: support database development, assess system weaknesses and make necessary improvements, perform backup procedures, test systems for efficiency and security, etc.
