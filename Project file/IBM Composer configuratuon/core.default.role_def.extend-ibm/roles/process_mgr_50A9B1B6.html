Роль: Process Manager
This role is responsible for the general management of the organization's process assets and for managing the strategic change required to adopt new process and tools.

Основное описание

The Process Manager manages the software capability improvement program. This role has these responsibilities:

  • ensuring that program and project activities are carried out according to specified objectives;
  • allocating resources to improvement projects;
  • planning and developing program activities;
  • monitoring process improvements;
  • performing process assessments of the organization's software development capabilities;
  • reporting to executive sponsors for schedule, budget, and quality of program elements;
  • escalating decisions to executive sponsors as necessary.
