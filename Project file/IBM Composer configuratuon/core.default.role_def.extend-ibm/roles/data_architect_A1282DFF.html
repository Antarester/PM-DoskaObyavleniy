Роль: Data Architect
This role is responsible for architecting the structure for storing and managing a solution's data.
Наборы ролей: Architecture
Взаимосвязи
Основное описание

The Data Architect is responsible for the design of the elements required to structure the data aspects of a solution, including databases, file systems, and storage management.

The Data Architect creates solution-related taxonomies and business views of data that are mapped to user business needs.

Персонал
Квалификация
  • Basic database design
  • Proficient data modeling
  • Detailed understanding of industry information models and approaches related to solution
  • Detailed understanding of overall technical goals of the business and the current information models used (XML, structured and unstructured)
  • Detailed understanding of relevant operating system for architecting appropriate physical design of storage and file systems
  • Proficient creation of database queries
  • Expert creation of information taxonomies
